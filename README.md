# BinaryGameBackEnd

Backend dla gry z zagadkami binarnymi. Poniżej najkrótsza instrukcja uruchomienia projektu lokalnie.

Backend: https://github.com/GryczonJ/BinaryGameBackEnd.git
Frontend: https://github.com/nimoCreator/BinaryGame-Frontend.git

## Wymagania
- Python 3.10+
- Zależności zainstalowane z `requirements.txt`

## Szybki start
1. (Opcjonalnie) utwórz i aktywuj wirtualne środowisko.
2. Zainstaluj zależności:
	- `pip install -r requirements.txt`
3. Uruchom aplikację:
	- `python -m uvicorn main:app --reload`

Po uruchomieniu serwer będzie dostępny lokalnie (domyślnie na porcie 8000).
